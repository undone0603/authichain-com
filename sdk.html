AuthiChain JavaScript SDK

Integrate product authentication into your app in minutes. Works in Node.js, browsers, and edge runtimes.

npm install @authichain/sdk
TypeScript Ready ESM & CJS Node 18+ Edge Runtime Browser Compatible

Quick Start

import AuthiChain from '@authichain/sdk';

const client = new AuthiChain({
  apiKey: process.env.AUTHICHAIN_API_KEY,
  // Optional: baseUrl defaults to https://authichain.com/api
});

// Issue a certificate
const cert = await client.certificates.issue({
  productId: 'prod_abc123',
  metadata: { color: 'midnight black', size: 'M' }
});
console.log(cert.txHash); // 0x...

// Verify a product (no API key needed)
const result = await client.verify('prod_abc123');
console.log(result.authentic); // true

const client = new AuthiChain({ apiKey: 'ac_live_...' });

SDK Methods

client.products.create(data)
Register a new product for authentication
client.products.get(id)
Retrieve product details and status
client.certificates.issue(data)
Issue a blockchain certificate
client.certificates.get(id)
Retrieve certificate with blockchain proof
client.verify(id)
Verify any product, cert, or tx hash
client.batch.authenticate(ids)
Bulk authenticate multiple products
client.analytics.get(opts)
Retrieve scan and verification analytics
client.webhooks.register(url, events)
Register a webhook endpoint for events

Batch Authentication

const results = await client.batch.authenticate([
  { name: 'Item #001', sku: 'SKU-001' },
  { name: 'Item #002', sku: 'SKU-002' },
  // ... up to 1,000 items per request
]);
results.forEach(r => console.log(r.certId, r.txHash));

Webhooks

await client.webhooks.register(
  'https://yourapp.com/webhooks/authichain',
  ['product.verified', 'certificate.issued', 'scan.recorded']
);

TypeScript

The SDK ships with full TypeScript types. No @types package needed.

import AuthiChain, { Certificate, VerifyResult } from '@authichain/sdk';

const cert: Certificate = await client.certificates.issue({ productId: 'prod_abc123' });
const result: VerifyResult = await client.verify('prod_abc123');
