AuthiChain API Documentation

The AuthiChain REST API lets you create and manage product authentications, issue blockchain certificates, and verify product legitimacy programmatically.

Base URL: https://authichain.com/api

Authentication

All API requests require an API key passed via the x-api-key header. Get your API key from the AuthiChain dashboard.

curl -H "x-api-key: YOUR_API_KEY" \
  https://authichain.com/api/health

Quickstart

Authenticate a product in 3 steps:

// 1. Create a product
POST /api/product
{ "name": "Artisan Bag #001", "brand": "LuxBrand", "sku": "LB-001" }

// 2. Issue certificate
POST /api/certificate
{ "productId": "prod_abc123" }

// 3. Verify (consumer)
GET /api/verify?id=prod_abc123

Products

POST/api/product
Register a new product for authentication
GET/api/product?id={productId}
Retrieve product details and authentication status

Certificates

POST/api/certificate
Issue a blockchain certificate for a product. Returns tx hash and certificate ID.
GET/api/certificate?id={certId}
Retrieve certificate details including blockchain proof

Verify

GET/api/verify?id={productId|certId|txHash}
Public endpoint. Verify any product by ID, certificate ID, or transaction hash. No API key required.

Batch

POST/api/batch
Authenticate multiple products in a single request. Growth plan and above.

Webhooks

Register webhook endpoints to receive real-time events: product.created, certificate.issued, product.verified, scan.recorded.

POST/api/webhook
Register a new webhook endpoint

SDK

Install the AuthiChain JavaScript SDK for easier integration. See the SDK documentation for full reference.

npm install @authichain/sdk

import AuthiChain from '@authichain/sdk';
const client = new AuthiChain({ apiKey: 'YOUR_API_KEY' });
const cert = await client.certificates.issue({ productId: 'prod_abc123' });
